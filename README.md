# Apex Home Services Mockups

This project contains the frontend mockups for Apex Home Services, built with React and Vite.

## Mockups Included
1. **Homepage**: Premium, Apple-style landing page.
2. **AI Assistant Chat**: Interactive chat interface for job requests.
3. **Booking Form**: Step-by-step service scheduling form.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **View Mockups**
   - Homepage: `http://localhost:5173/`
   - AI Assistant: `http://localhost:5173/chat`
   - Booking Form: `http://localhost:5173/book`

## Tech Stack
- **React**: UI Library
- **Vite**: Build Tool
- **React Router**: Navigation
- **Lucide React**: Icons
- **Vanilla CSS**: Custom premium styling (Apple/Stripe aesthetic)
